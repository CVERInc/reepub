Reepub

把你的紙，裝幀成私人電子書庫 Bind your own paper into a personal library

完全離線的 macOS 工具，用 Apple Vision OCR 把你擁有的掃描 PDF，裝幀成乾淨、可重排、已驗證的 EPUB3 電子書。 A 100% offline macOS tool that turns scanned PDFs you own into clean, reflowable, validated EPUB3 books — powered by Apple Vision OCR.

僅供處理你擁有、或有權數位化的檔案 —— 你的筆記、信件、手稿、公有領域藏書。一切都在你自己的 Mac 上進行，永遠不上傳。 For documents you own or have the right to digitize — your notes, letters, manuscripts, public-domain works. Everything runs on your own Mac; nothing is ever uploaded.

為什麼用 ReepubWhy Reepub

多數「PDF 轉 EPUB」工具不是上傳雲端、按頁收費，就是吐出一堆圖片拼成的固定版面。Reepub 不一樣。 Most "PDF to EPUB" tools upload to a cloud, charge per page, or spit out image-glued fixed layouts. Reepub is different.

🔒

完全離線100% offline

沒有 API key、沒有訂閱、沒有網路。你的文件永遠不離開這台 Mac。No API keys, no subscriptions, no internet. Your files never leave your Mac.

🧠

Apple Vision OCRApple Vision OCR

在 Apple Silicon 上由 Neural Engine 加速，又快又準。Neural-Engine-accelerated on Apple Silicon — fast and accurate.

📖

真正可重排Truly reflowable

文字重組成真正的段落與章節，在任何螢幕尺寸自動重排，不是凍結的頁面圖片。Text is rebuilt into real paragraphs and chapters that reflow on any screen — not frozen page images.

🀄

繁中 + 英文zh-Hant + English

內建繁體中文與英文辨識（zh-Hant + en-US）。Traditional Chinese and English recognition out of the box.

✅

驗證過的 EPUB3Validated EPUB3

每本書交回前都通過內建、零依賴的結構驗證器；自動封面、章節、圖文混排頁。Every book passes a built-in, dependency-free validator. Auto cover, chapters, hybrid text/image pages.

⚖️

MIT 開源MIT open source

自包含、可自由 fork、永遠免費。Self-contained, forkable, free forever.

開始使用Get started

Reepub 從原始碼建置，只需 Xcode Command Line Tools，不需要完整的 Xcode。 Reepub builds from source with just the Xcode Command Line Tools — no full Xcode needed.

推薦Recommended

原生 macOS AppNative macOS app

選一份 PDF（或拖進視窗），讓 OCR 跑完，填入書名與作者，按「存成 EPUB…」。OCR、組裝與驗證全在 App 內完成。Pick a PDF (or drag it in), let OCR run, set a title and author, then "Save EPUB…". OCR, assembly and validation all happen inside the app.

# 建置並開啟原生 Appbuild & open the native app
git clone https://github.com/CVERInc/reepub.git
cd reepub
make app
open macos/build/Reepub.app
B

本機網頁介面Local web UI

在 localhost 開啟拖放介面，轉換日誌即時串流。A drag-and-drop UI on localhost with a live-streaming conversion log.

make build   # 編譯 Swift OCR CLIcompile the Swift OCR CLI
npm start    # http://localhost:30232
